A web-based interactive drawing application built with WebGL that allows users to create art using points (squares), triangles, and circles.

## Features

- **Interactive Canvas:** Draw shapes by clicking and dragging on the canvas
- **Multiple Shape Types:** Choose between Squares (points), Triangles, or Circles
- **Color Control:** RGB sliders to customize shape colors
- **Size Control:** Adjustable shape size slider
- **Circle Segments:** Control the smoothness of circles with segment count
- **Clear Canvas:** Reset and start fresh anytime
- **Reference Image:** View the original hand-drawn picture (house with cloud and sun) on the right side

## How to Use

1. Select a drawing mode (Squares, Triangles, or Circles)
2. Adjust the RGB color sliders to choose your color
3. Set the shape size and circle segment count
4. Click and drag on the canvas to draw
5. Use "Clear Canvas" to start over
