# CSE 160 Assignment 1 - Interactive Drawing Application

A fun web-based drawing application built with WebGL that lets you create art using points, triangles, and circles.

## Features

- **Draw with different shapes**: Choose between squares (points), triangles, or circles
- **Customize your colors**: Use RGB sliders to pick any color you want
- **Control size and detail**: Adjust shape size and circle segment count
- **Click and drag**: Draw continuously by holding down and moving your mouse
- **Clear canvas**: Start fresh anytime with the clear button
- **Draw a picture**: Click the "Draw Picture" button to see a pre-made scene

## How to Use

Open `asgn1.html` in a modern web browser (Chrome, Firefox, or Edge recommended). Click on the canvas to draw shapes, use the controls to customize your drawing, and have fun creating!
