# CSE160 Assignment 2 Blocky 3D Animal
A class assignment to create an interactive program centered around a 3D animal.

Open `asgn2.html` in a modern browser to run the project.
The creature is built from cubes and a pyramid (hat), with sliders for head, tongue, legs, and a three-joint tail.
Use the camera sliders or drag on the canvas to rotate the view; hold Shift and click for a poke reaction.
Scripts: `asg2.js` (main logic), `cube.js`, `pyramid.js`; dependencies live in `lib/`.
